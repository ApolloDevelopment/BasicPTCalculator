  • getHypotenuse(a, b); // input the value of the A and B legs to get the length of leg C.
  • getMissingLeg(b, c); // input the value of the B (or A) and C legs to get the length of leg A (or B).
  • getRelation(a, b, c); // states whether the triangle is obtuse, acute, or right based on the side lengths given to the function.
  • firstLine(); // logs the pre-loaded line to the console that states the available functions
  • radical(a, b) // gets the final value of a complex radical
  • (Tip: if a side length is a radical, use "(# ** 0.5)" as that value and whatever number is in the radical replaces the '#' sign)


  • Have a complex radical and already have the squared numbers?
  • getHsquared(a, b) // gets the hypotenuse with the already-squared numbers
  • getMLsquared(b, c) // gets the missing leg with the already-squared numbers
  • getRsquared(a, b, c) // gets the relationship of the triangle with the already-squared numbers
